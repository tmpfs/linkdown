$0
==

Link manipulation tool.

## Commands

* `info: info, i`: Print link information.

## Options

* `json: -j, --json`: Print output as JSON.
* `--log-level=[level]`: Set the log level.
* `--depth=[int]`: Maximum depth to recurse.

## Example

Print links:

```
$0 info http://example.com
```
